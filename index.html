Custom Programming Language

>

[Github Link]
❮ ❯

I developed a recursive programming language from scratch in C++. I created a custom grammar and structure for the programming language and implmeented the front and back end of the language.

I worked on this project using C++, LLVM, ANTLR, and Clang. In the language I created, I implemented variables, procedures, functions, return statements, importing libraries, importing external variables and functions, implicit typing, recursion, conditional statements, and loops. To import external libraries, the user of the language is able to link the file with the corresponding library or external file after compiling the program. I defined a Context Free Grammar (CFG) and used ANTLR to create a parse tree for input files. I then used this parse tree along with LLVM to compile this tree into x86_64 assembly code which can be run.

Skills: Assembly, Programming Language Theory, C++
Tools: C++, Clang, LLVM, ANTLR, Linux, Git

Hospital Application

>

In collaboration with Brigham and Women's Hospital

[Github Link]

This project was completed while following Agile Methodologies on a team of 10 members. The finished application has the ability to keep track of medical equipment, and track different workflows necessary to Brigham and Women's Hospital for multiple users with configurable permissions.

These workflows include service requests, which are handled via Java querying an Apache Derby SQL server using JDBC. The SQL server is also used to maintain persistent data between application runs. The application also includes a custom pathfinding functionality spanning nultiple floors of the hospital, where users can select rooms, and be given the shortest path between these locations. The application allows creation of accounts for general users and administrative users, and allows color scheme choices as well as facial recognition login to be saved for each user. The application also interacts with the Google Maps API to proivde printable directions to Brigham and Women's Hospital from a specified location. For the duration of the project, I acted as the Scrum Maaster, and facilitated daily scrums.

Skills: Agile, SQL, Java, Full Stack
Tools: Java, SQL, HTML, CSS, Gradle, Guice, Apache Derby, Git
